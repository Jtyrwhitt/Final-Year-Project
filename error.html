Types of Errors that occur in python

Error Type Description Example
Syntax Error Occurs when Python encounters incorrect syntax (rules of the language are broken). 
    print("Hello World"   # Missing closing parenthesis
Runtime Error Occurs when the code is syntactically correct but encounters an issue during execution. 
    x = 5 / 0   # Division by zero (ZeroDivisionError)
Logical Error Occurs when the program runs without crashing but produces incorrect results. 
    def add_numbers(a, b):
        return a - b  # Incorrect operation (should be a + b)
ValueError Occurs when a function receives an argument of the right type but an inappropriate value. 
    int("hello")  # Cannot convert a string to an integer
NameError Occurs when trying to access a variable that has not been defined. 
    print(my_variable)  # my_variable is not defined
IndexError Occurs when trying to access an index that does not exist in a list or string. 
    my_list = [1, 2, 3]
    print(my_list[5])  # Index 5 is out of range
ModuleNotFoundError Occurs when trying to import a module that is not installed or does not exist. 
    import non_existent_module  # This module does not exist
TypeError Occurs when an operation is performed on an incorrect data type. 
    print("Hello" + 5)  # Cannot concatenate string and integer
