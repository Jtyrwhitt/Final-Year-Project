Variables

What is a variable?

A variable is used to store data that will get used in the program. An variable can consist of a Integer, String, Boolean, Lists and more other data types.

An equal sign is used to assign an value to a variable which can be updated after the initial set value to a new value when needed.

Variable Names

The variable name can consist of letters, numbers and a underscore character.

Variable naming rules

  • Variable must start with a letter or underscore character and must not start with a number.
  • Variable can only contain alpha numeric characters and underscores (A-Z, 0-9, _)
  • Variables are case sensitives

Variable Types

  • Integer (int): Whole numbers, eg a = 5
  • Float (float): Decimal numbers, eg b = 3.14
  • String (str): Text, eg name = "John"
  • Boolean (bool): True or False, eg is_active = True
