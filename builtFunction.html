Python Built-in Functions

Python provides many built-in functions that perform common operations. Below is a table of some frequently used built-in functions along with their descriptions and examples.

Common Built-in Functions

Function Description Example
print() Displays output on the console. 
print("Hello, World!")
len() Returns the length of an object (string, list, etc.). 
print(len("Python"))  # Output: 6
type() Returns the type of an object. 
print(type(42))  # Output: <class 'int'>
sum() Returns the sum of all elements in an iterable. 
print(sum([1, 2, 3]))  # Output: 6
max() Returns the maximum value in an iterable. 
print(max(10, 20, 30))  # Output: 30
min() Returns the minimum value in an iterable. 
print(min([4, 2, 8]))  # Output: 2
round() Rounds a number to a specified number of decimals. 
print(round(3.14159, 2))  # Output: 3.14
abs() Returns the absolute value of a number. 
print(abs(-5))  # Output: 5
sorted() Returns a sorted list of an iterable. 
print(sorted([3, 1, 4, 1]))  # Output: [1, 1, 3, 4]
input() Takes user input from the console. 
name = input("Enter your name: ")
