Recursion

In Python, functions can call themselves, a concept referred to as "recursion." Recursion is a technique that enables a function to be divided into smaller, more manageable parts, allowing for more efficient processing.

Basic Syntax

Recursive functions are typically designed to work with their parameters in two ways:

  • A recursive call to the function, where the parameters are reduced or simplified.
  • One or more base cases that halt the recursion, returning data that can then be combined back through the chain of
    recursive calls.

Examples

In this example, the countDown function recursively calls itself, decrementing the number by 1 each time. When the number
reaches zero, it prints "Done!" and stops the recursion.
